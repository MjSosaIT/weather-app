# Weather App

## Description
This is a simple Weather App that allows users to search for real-time weather information using the OpenWeatherMap API. It displays temperature, weather conditions, and humidity for the entered city.

## Features
- Search for weather by city name
- Displays temperature, weather conditions, and humidity
- Provides error handling for invalid city names

## Installation
1. Download the project files.
2. Replace `'YOUR_API_KEY'` in `script.js` with your OpenWeatherMap API key.
3. Open `index.html` in a web browser to view the app.
